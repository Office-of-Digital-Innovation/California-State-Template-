Usage

When and how to use it

Use social media icons in the site footer. These icons should link to your official social media accounts.

The available social media icons include:

  • Facebook
  • Twitter
  • Instagram
  • YouTube
  • GitHub
  • LinkedIn
  • Email

Browse our State Web Template icons.

How not to use it

Do not include more than one social media account per platform. Do not link to social media accounts that are not updated or monitored regularly.
